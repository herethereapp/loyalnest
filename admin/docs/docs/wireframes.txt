Explanation of Wireframes
Merchant Dashboard:
- Welcome Page: Guides setup with tasks and messages (US-MD1, Phase 1).
- Points Program: Configures earning/redemption rules, branding, status (US-MD2, Phase 1).
- Referrals Program: Manages Klaviyo/Postscript notifications, social sharing (Phase 2), referral status, and paginated codes (US-MD3, US-CW7, Phase 1–2).
- Customers: Lists customers with AES-256 encrypted email, points, RFM, search, pagination, and CSV import (US-MD4, US-BI3, Phase 1–3).
- Analytics: Shows metrics, Chart.js bar charts, RFM segment previews, and CSV export (US-MD5, US-MD12, Phase 1–3).
- Settings: Configures store details, billing, RFM (Phase 2), developer toolkit, and language with i18next (Phase 3) (US-MD6, US-MD10).
- On-Site Content: Manages loyalty page, rewards panel, launcher, popups, checkout extensions (Phase 2), sticky bar, and post-purchase widget with i18next (US-MD7, US-MD16, Phase 1–3).
- Notification Templates: Configures multilingual email/SMS templates via Klaviyo/Postscript (US-MD8, Phase 2).
- VIP Tiers: Sets up tiers and multilingual notifications (US-MD7, Phase 2).
- Activity Logs: Displays paginated logs for points, referrals, VIP changes (US-MD9, Phase 2).
- Bonus Campaigns: Configures campaigns with schedules and caching (US-MD10, US-BI4, Phase 3).
- Rate Limit Monitoring: Shows Shopify API and integration status (US-AM11, Phase 2).

Customer Widget:
- Single interface with points balance, redemption (including campaign discounts), referral popup/status, RFM nudges, VIP tiers, gamification, GDPR form, and language dropdown (US-CW1–CW12, Phase 1–3).
- Includes order cancellation notifications (US-CW6), error states, and high-contrast mode.
- Uses Tailwind CSS for mobile responsiveness and i18next for multilingual support.

Admin Module:
- Overview: Displays merchant count, points, ROI, and Chart.js RFM charts (US-AM1, Phase 1).
- Merchants: Lists merchants with points adjustment and suspension (US-AM2, US-AM10, Phase 1).
- Admin Users: Manages admin accounts with RBAC (US-AM4, Phase 1).
- Logs: Shows paginated API and audit logs (US-AM5, Phase 1).
- Plan Management: Handles plan changes (US-AM7, Phase 2).
- Integration Health: Monitors Shopify, Klaviyo, Postscript, Square (US-AM8, Phase 2–3).
- RFM Config Management: Configures RFM thresholds with Chart.js preview (US-AM9, Phase 2).
- RFM Segment Export: Exports segments with Chart.js bar chart (US-AM11, Phase 3).
- GDPR Requests: Manages data requests/redactions with 90-day retention display (US-AM6, Phase 1).
- Rate Limit Monitoring: Shows API and integration rate limits (US-AM11, Phase 2).

Notes:
- ASCII Limitations: Uses simple borders and placeholders ([Chart], [Button: ...]). Chart.js bar charts specified for RFM and metrics.
- Polaris Compliance: Merchant Dashboard uses Polaris components (Tabs, FormLayout, Button) for Shopify UX.
- Tailwind CSS: Customer Widget ensures responsive, mobile-friendly design.
- Accessibility: ARIA labels on all interactive elements, keyboard navigation for dropdowns/modals, high-contrast mode, and screen reader prompts for charts.
- Scalability: Pagination/lazy loading for `Customers`, `Referral Codes`, `Logs`, `GDPR Requests`. Redis caching (e.g., `points:{customer_id}`, `rfm:preview:{merchant_id}`).
- Security: Shopify OAuth (Merchant Dashboard, Customer Widget), RBAC (Admin Module), AES-256 encryption (`customers.email`, `reward_redemptions.discount_code`).
- Multilingualism: JSONB fields (`email_templates.body`, `nudges.title`, `merchants.language`) with `CHECK ?| ARRAY['en', 'es', 'fr']`, i18next integration.
- Analytics: PostHog events (e.g., `points_history_viewed`, `gdpr_request_submitted`) aligned with `Sequence Diagrams.txt`.
